# MODULE 6 : LES 3 ÉLÉMENTS DE L'ACHAT

## Pour que quelqu'un achète, 3 éléments doivent être présents :

1. **L'IDENTITÉ** (La vente identitaire)
2. **L'ÉMOTIONNEL** (Le système limbique)
3. **LA LOGIQUE** (La rationalisation)

Si UN SEUL manque → Pas de vente.

---

## ÉLÉMENT #1 : L'IDENTITÉ

### Le Principe Fondamental

> **Les gens achètent l'IDENTITÉ, pas le produit.**

Personne n'achète un accompagnement.
Tout le monde achète l'identité que l'accompagnement permet d'atteindre.

### Exemples

| Ce que tu vends | Ce qu'ils achètent |
|-----------------|-------------------|
| Programme de perte de poids | L'identité de quelqu'un d'athlétique |
| Formation business | L'identité de quelqu'un de successful |
| Coaching closing | L'identité d'un top closer |

### Ce Qui Se Passe Dans Leur Tête

**En surface :** "J'achète un programme"

**En profondeur :**
- L'identité souhaitée
- Les valeurs partagées
- Le besoin d'appartenance
- L'image qu'ils veulent renvoyer

### Comment Vendre sur l'Identité

**1. Mets le prospect dans l'identité de quelqu'un qui PASSE À L'ACTION**

Dès le début du call, colle-lui des étiquettes :
```
"Tu as l'air super motivé"
"T'es quelqu'un de déterminé"
"Je vois que t'es ambitieux"
```

**Pourquoi ça marche :**
Le prospect veut être COHÉRENT avec l'étiquette que tu lui as mise.
S'il est "déterminé", il agira comme quelqu'un de déterminé.

**2. Projette-le dans son identité souhaitée**

```
"Imagine-toi dans 6 mois, tu fais 50K/mois..."
"Le [son prénom] qui a atteint ses objectifs..."
"Quand tu seras cette version de toi..."
```

**3. Utilise le biais de cohérence**

Fais-lui répéter plusieurs fois qu'il est déterminé, qu'il passe à l'action, etc.

Plus il le dit → Plus il agira en cohérence avec ça.

### L'Anti-Pattern

❌ Ne jamais lui coller une identité NÉGATIVE :
```
"Tu as l'air hésitant"
"Tu as du mal à te décider"
```
→ Il agira en cohérence avec ça aussi !

## ÉLÉMENT #2 : L'ÉMOTIONNEL

### Le Système Limbique

Rappel : Les gens prennent des décisions avec leurs ÉMOTIONS (Système 1), pas avec leur logique (Système 2).

### Comment Activer le Système Limbique

**1. Créer une TENSION émotionnelle**

Le call doit avoir des pics émotionnels :
- Des moments POSITIFS (enthousiasme, espoir)
- Des moments NÉGATIFS (urgence, peur)

```
CALL PLAT (pas de vente) :
────────────────────────────

CALL AVEC TENSION (vente) :
    /\    /\
   /  \  /  \
──/    \/    \───
```

**2. Le Pouvoir du Contraste**

En alternant positif/négatif :
- Le positif a PLUS d'impact
- Le négatif a PLUS d'impact
- Le prospect est dans un "flow émotionnel"

**3. Ne Pas Avoir un Call "Flat"**

Si ton cardiogramme émotionnel est plat → Pas de vente.

Pourquoi ? Parce qu'un call plat sollicite le cortex préfrontal (logique), pas le système limbique (émotions).

### Les Questions Émotionnelles

❌ Questions plates :
```
"Comment ça va ?"
"Tu en es où ?"
"C'est quoi ton objectif ?"
```

✅ Questions émotionnelles :
```
"Comment tu te SENS par rapport à ça ?"
"Qu'est-ce que ça te fait de voir des gens de ton âge réussir alors que toi..."
"Comment ça t'IMPACTE au quotidien ?"
```

### La Douleur Décisionnelle

Rappel du Module 4 :
- **Inconfort** = "C'est chiant" → Pas de décision
- **Douleur décisionnelle** = "Je ne peux PLUS continuer" → Décision

Tu dois amener le prospect de l'inconfort à la douleur décisionnelle.

### Créer l'Urgence RÉELLE (Pas Artificielle)

❌ Fausse urgence :
```
"Il reste 4 places"
"Les portes ferment demain"
```

✅ Vraie urgence :
```
"Chaque jour où tu ne fais rien, c'est de l'argent que tu laisses sur la table"
"Depuis combien de temps tu repousses ? Et qu'est-ce que ça t'a apporté ?"
"Dans 6 mois, tu seras au même endroit si tu ne changes rien"
```

## ÉLÉMENT #3 : LA LOGIQUE

### Le Rôle de la Logique

La logique ne fait PAS prendre la décision.
Elle JUSTIFIE la décision qui a déjà été prise émotionnellement.

### Quand Utiliser la Logique ?

**APRÈS avoir activé l'émotionnel.**

Structure :
1. Identité ✓
2. Émotionnel ✓
3. → Logique (Pitch)

### Le Pitch : Rationaliser la Décision

Le pitch sert à :
- Rassurer le prospect
- Lui donner des arguments pour SE justifier
- Lui donner des arguments pour justifier auprès des AUTRES

### Structure du Pitch Efficace

**1. Personnalisé**

Ne pitch que ce qui correspond à SES besoins.

S'il veut du 1-1 → Insiste sur le 1-1
S'il veut de l'autonomie → Insiste sur les ressources

**2. Clarté**

Le pitch doit SIMPLIFIER, pas compliquer.

Choix entre 100 plats = Difficile
Choix entre 2 plats = Facile

Pareil pour ton pitch : Moins c'est mieux.

**3. Bénéfices > Caractéristiques**

| Caractéristique | Avantage | Bénéfice |
|-----------------|----------|----------|
| "10 modules" | "Ça peut t'aider" | "Tu pourras faire X que tu voulais" |
| → Inquiétude prix | → Objection "je réfléchis" | → Décision d'achat |

**4. Durée : 6-8 minutes MAX**

Au-delà, tu perds le prospect.

### Relier au Résultat

Chaque élément du pitch doit être relié à UN RÉSULTAT CONCRET pour le prospect.

```
"On a [caractéristique]"
❌

"On a [caractéristique], ce qui va te permettre de [résultat qu'il veut]"
✅
```

### Projection Identitaire dans le Pitch

❌ "SI tu atteins X..."
✅ "QUAND tu auras atteint X..."

Présuppose le succès. Ça renforce l'engagement.

---

## Le Flow Complet

```
1. IDENTITÉ
   → Mettre le prospect dans l'identité de quelqu'un qui passe à l'action
   → Projeter dans l'identité souhaitée

2. ÉMOTIONNEL
   → Créer la tension émotionnelle
   → Pics positifs et négatifs
   → Douleur décisionnelle

3. LOGIQUE
   → Pitch personnalisé
   → Rationalisation de la décision
   → Arguments pour se justifier

= VENTE
```

---

## Pourquoi C'est Différent du Closing "Classique"

### Closing Classique
```
Situation actuelle → Situation désirée → Pitch → Close
```
= Principalement LOGIQUE = Peu efficace

### Closing Avancé
```
Identité → Émotionnel → Logique
```
= COMPLET = Très efficace

## Checklist des 3 Éléments

Avant de pitcher, vérifie :

### Identité
- [ ] J'ai collé des étiquettes positives (motivé, déterminé...)
- [ ] J'ai projeté le prospect dans son identité souhaitée
- [ ] J'ai utilisé le biais de cohérence

### Émotionnel
- [ ] Le call a eu des pics émotionnels
- [ ] J'ai créé de la tension (positif/négatif)
- [ ] Le prospect a ressenti une douleur décisionnelle
- [ ] L'urgence est réelle, pas artificielle

### Logique
- [ ] Mon pitch est personnalisé
- [ ] Je n'ai pitché que ce qu'il veut
- [ ] J'ai parlé en bénéfices, pas en caractéristiques
- [ ] Mon pitch a duré moins de 8 minutes

### La Formule
```
Identité + Émotionnel + Logique = Vente Inévitable
```

### Ce Que Tu Dois Retenir

1. Les gens achètent une IDENTITÉ
2. La décision est ÉMOTIONNELLE
3. La logique ne fait que JUSTIFIER
4. Les 3 doivent être présents
