# MODULE 22 : CHECKLISTS OPÉRATIONNELLES

## OBJECTIF
Checklists prêtes à l'emploi pour ne jamais oublier un élément clé à chaque étape du processus de closing.

---

# CHECKLIST 1 : AVANT CHAQUE JOURNÉE DE CALLS

## Préparation physique et mentale
- [ ] Sommeil suffisant (7-8h)
- [ ] Douche / rafraîchissement
- [ ] Tenue appropriée (même si caméra off, ça impacte ta posture)
- [ ] Espace de travail rangé
- [ ] Verre d'eau à portée

## Préparation technique
- [ ] Connexion internet testée
- [ ] Micro testé
- [ ] Caméra testée et bien positionnée
- [ ] Éclairage vérifié (visage bien éclairé)
- [ ] Arrière-plan neutre/professionnel
- [ ] Notifications désactivées
- [ ] Téléphone en silencieux

## Préparation mentale
- [ ] Objectifs du jour écrits
- [ ] Rappel des outcomes de chaque phase
- [ ] Posture d'autorité visualisée
- [ ] Affirmations récitées (optionnel)
- [ ] Respiration profonde × 3

## Matériel prêt
- [ ] CRM/Tableau de suivi ouvert
- [ ] Formulaires des prospects consultés
- [ ] Notes sur les prospects du jour prises
- [ ] Lien de paiement prêt
- [ ] Enregistrement activé

---

# CHECKLIST 2 : AVANT CHAQUE CALL (15 min avant)

## Recherche prospect
- [ ] Formulaire relu en détail
- [ ] Profil réseaux sociaux consulté (si disponible)
- [ ] Niveau de conscience identifié
- [ ] Type de prospect probable identifié

## Préparation stratégique
- [ ] Objectifs du prospect notés
- [ ] Douleurs potentielles anticipées
- [ ] Objections probables listées
- [ ] Reframes prévues pour chaque objection
- [ ] Questions clés préparées

## État mental
- [ ] Call précédent mentalement fermé
- [ ] Énergie au bon niveau
- [ ] Détaché du résultat
- [ ] Curiosité sincère pour le prospect

## Dernière minute
- [ ] Nom du prospect mémorisé (prononciation correcte)
- [ ] Lien de call vérifié
- [ ] Chrono mental réinitialisé

---

# CHECKLIST 3 : PENDANT LE CALL - PHASE PAR PHASE

## Phase 1 : Ouverture (0-5 min)
- [ ] Accueil chaleureux mais professionnel
- [ ] Caméra allumée (des deux côtés)
- [ ] Question d'ouverture posée (pas de small talk forcé)
- [ ] Contexte de l'appel rappelé
- [ ] Garde du prospect baissée

**Outcome atteint ?** [ ] Oui [ ] Non → Ne pas avancer si non

## Phase 2 : Diagnostic - Situation (5-10 min)
- [ ] Situation actuelle clarifiée
- [ ] Chiffres concrets obtenus
- [ ] Contexte personnel compris
- [ ] Ce qu'il a déjà essayé identifié

**Outcome atteint ?** [ ] Oui [ ] Non → Ne pas avancer si non

## Phase 3 : Diagnostic - Douleurs (10-20 min)
- [ ] Problème principal identifié
- [ ] Conséquences du problème creusées
- [ ] Impact émotionnel verbalisé par le prospect
- [ ] Urgence de changer établie
- [ ] Pics émotionnels créés

**Outcome atteint ?** [ ] Oui [ ] Non → Ne pas avancer si non

## Phase 4 : Objectifs (5-10 min)
- [ ] Objectif concret défini (chiffres)
- [ ] Timeline souhaitée établie
- [ ] Raison émotionnelle profonde identifiée
- [ ] Projection dans le futur faite

**Outcome atteint ?** [ ] Oui [ ] Non → Ne pas avancer si non

## Phase 5 : Anticipation objections (5-10 min)
- [ ] Investissements passés explorés
- [ ] Ce qui l'a retenu identifié
- [ ] Peurs principales découvertes
- [ ] Croyances limitantes adressées
- [ ] Budget approximatif sondé

**Outcome atteint ?** [ ] Oui [ ] Non → Ne pas avancer si non

## Phase 6 : Transition vers pitch
- [ ] Résumé de la situation fait
- [ ] Confirmation que le prospect est prêt
- [ ] Permission de présenter obtenue

## Phase 7 : Pitch (6-8 min)
- [ ] Transition fluide
- [ ] Vision du résultat peinte
- [ ] Mécanisme unique expliqué
- [ ] Composants reliés à SES objectifs
- [ ] Preuves sociales pertinentes
- [ ] Questions avant prix proposées

**Pitch personnalisé ?** [ ] Oui [ ] Non

## Phase 8 : Annonce du prix
- [ ] Prix annoncé avec certitude
- [ ] Silence maintenu après
- [ ] Réaction observée sans réagir

## Phase 9 : Traitement d'objections (si nécessaire)
- [ ] Objection écoutée entièrement
- [ ] Vraie objection identifiée (argent/peur ?)
- [ ] Reframe appropriée utilisée
- [ ] Nouvelle objection traitée si apparue
- [ ] Isolation de l'objection faite

## Phase 10 : Closing
- [ ] Décision demandée clairement
- [ ] Paiement proposé
- [ ] Prochaines étapes expliquées

# CHECKLIST 4 : APRÈS CHAQUE CALL

## Dans les 2 minutes qui suivent

### Si CLOSÉ :
- [ ] Félicitation envoyée au prospect
- [ ] Lien de paiement envoyé (si pas fait en call)
- [ ] Confirmation de réception attendue
- [ ] Prochaines étapes envoyées
- [ ] CRM mis à jour (statut : closé)
- [ ] Commission notée

### Si NON CLOSÉ :
- [ ] Type de non identifié (objection / pas intéressé / R2)
- [ ] Raison de non-closing notée
- [ ] CRM mis à jour
- [ ] Follow-up planifié (si applicable)
- [ ] Date de relance fixée

## Auto-feedback rapide (2 min)
- [ ] Qu'est-ce qui a bien marché ?
- [ ] Qu'est-ce que j'aurais pu faire différemment ?
- [ ] Quelle compétence travailler ?

## Reset mental
- [ ] Ce call est fermé mentalement
- [ ] Respiration profonde
- [ ] Prêt pour le call suivant

# CHECKLIST 8 : TRAITEMENT D'OBJECTION

## Étape 1 : Identifier
- [ ] Objection entendue complètement (pas coupé)
- [ ] Type identifié : [ ] Argent [ ] Peur [ ] Bullshit
- [ ] Vraie objection ou écran de fumée ?

## Étape 2 : Valider
- [ ] "Je comprends" ou équivalent utilisé
- [ ] Pas de justification immédiate
- [ ] Posture calme maintenue

## Étape 3 : Clarifier
- [ ] Question de clarification posée
- [ ] "C'est-à-dire ?" ou "Par rapport à quoi ?"
- [ ] Vraie raison obtenue

## Étape 4 : Isoler
- [ ] "Mis à part ça, tu te lancerais ?" posé
- [ ] Autres objections éliminées
- [ ] Focus sur UNE objection

## Étape 5 : Reframe
- [ ] Technique appropriée choisie
- [ ] Reframe délivrée calmement
- [ ] Question de validation posée

## Étape 6 : Avancer
- [ ] Objection résolue vérifiée
- [ ] Retour vers le closing
- [ ] Si nouvelle objection → retour étape 1

# CHECKLIST 9 : FOLLOW-UP

## Immédiatement après le call (si R2)
- [ ] Date de R2 fixée
- [ ] Récap envoyé au prospect
- [ ] Vidéo de pré-endoctrinement envoyée
- [ ] Rappel calendrier créé

## J-1 avant le R2
- [ ] Message de rappel envoyé
- [ ] Vocal personnalisé (optionnel)
- [ ] Vidéo supplémentaire envoyée

## Jour du R2
- [ ] Appel de confirmation (optionnel)
- [ ] Notes du call précédent relues
- [ ] Objections anticipées

## Si no-show
- [ ] Appel immédiat
- [ ] Message texte envoyé
- [ ] Nouveau RDV proposé
- [ ] Si pas de réponse → relance J+1

## Relance long terme (si pas closé)
- [ ] J+7 : Message de valeur
- [ ] J+14 : Témoignage client
- [ ] J+30 : Proposition de nouveau call
- [ ] Au-delà : Nurturing passif

---

# CHECKLIST 10 : OPTIMISATION SHOW-UP

## Avant réservation
- [ ] Formulaire de qualification complet
- [ ] Questions de commitment intégrées
- [ ] Créneaux limités proposés

## Après réservation (immédiat)
- [ ] Confirmation automatique envoyée
- [ ] Groupe WhatsApp créé
- [ ] Première vidéo d'endoctrinement envoyée
- [ ] Appel de qualification si possible

## J-1
- [ ] Rappel personnalisé (vocal > texte)
- [ ] Heure et lien confirmés
- [ ] Deuxième vidéo envoyée

## Jour J (matin si call l'après-midi)
- [ ] Dernier rappel
- [ ] "J'ai hâte de te parler" envoyé
- [ ] Lien renvoyé

## Si risque de no-show détecté
- [ ] Appel direct
- [ ] Reconfirmation de l'engagement
- [ ] Étiquette "tu n'es pas le genre de personne qui pose un lapin" placée

---

# MÉMO : LES 10 COMMANDEMENTS DU CLOSER

```
1. Tu prépareras chaque call comme si c'était le dernier
2. Tu n'avanceras pas sans avoir atteint l'outcome de chaque phase
3. Tu écouteras plus que tu ne parleras
4. Tu ne pitcheras jamais sans diagnostic complet
5. Tu traiteras la vraie objection, pas le bullshit
6. Tu maintiendras ta posture quoi qu'il arrive
7. Tu trackeras toutes tes données
8. Tu analyseras chaque call non closé
9. Tu t'entraîneras chaque semaine
10. Tu ne lâcheras jamais un prospect closable
```

La discipline fait la différence entre un closer moyen et un closer d'élite.
